# Login-Dashboard
# 🔐 Secure Portal (Login & Role-Based Dashboard)

A modern **front-end authentication system** with a sleek glassmorphism UI.
This project demonstrates a simple login flow with **role-based dashboards** (Admin & User) using **pure HTML, CSS, and JavaScript**.

---

## ✨ Features

* 🔑 Login authentication (client-side)
* 👤 Role-based access:

  * **Admin Dashboard**
  * **User Dashboard**
* 💾 Session handling using `sessionStorage`
* 🎨 Modern **Glassmorphism UI**
* 🌌 Animated gradient background
* ⚡ Fully responsive layout
* 🚀 No frameworks – built with vanilla JS

---

## 📂 Project Structure

```
📁 project-folder
├── index.html     # Main HTML structure
├── style.css      # UI styling (Glassmorphism + animations)
├── app.js         # Application logic (login + dashboard)
```

---

## 🧠 How It Works

* Predefined users are stored in JavaScript 
* On login:

  * Credentials are validated
  * User session is stored in `sessionStorage`
* Based on role:

  * Admin → Full dashboard access
  * User → Limited dashboard
* Logout clears session and redirects to login
